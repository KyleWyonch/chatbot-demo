Academic Paper Knowledge Graph Assistant

Interactive interface for exploring structured NLP research knowledge extracted from ACL Anthology papers
This assistant provides natural-language access to a knowledge graph constructed from thousands of computational linguistics research papers. The system automatically extracts relationships between datasets, models, tasks, and evaluation metrics across the literature and allows users to query research trends interactively.
Example questions you can ask:
  • Which datasets are used in machine translation research?
  • Which models appear most frequently in question answering papers?
  • What evaluation metrics are common in summarization?
Built using entity extraction, clustering, classification, and knowledge graph construction over ACL Anthology research papers
